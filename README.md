Duncan, EM, J Arrowsmith, C Bain, AC Broderick, J Lee, K Metcalfe, SK Pikesley, RTE Snape, E van Sebille and BJ Godley (2018), Marine Pollution Bulletin, 136, 334-340 (https://doi.org/10.1016/j.marpolbul.2018.09.019)
